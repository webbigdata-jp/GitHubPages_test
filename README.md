# llm-jp.github.io

### Public URL

- https://llm-jp.nii.ac.jp/

### ページ更新方法

- 主要ページ（概要・資料・メンバー・連絡先）を新たに追加する場合は、 `_data/navigation.yml` に追記したあと `_pages` に記事を作成する（ヘッダの permalink で URL を対応付ける）
- 投稿を追加する場合は、 `_posts` に日付入りのファイルを作成し `layout: single` をヘッダに記入しておく
